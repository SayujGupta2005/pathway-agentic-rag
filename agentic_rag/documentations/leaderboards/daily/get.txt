Get daily leaderboard.

**Rate limit:** This operation can be called up to 500 times per hour for regular users.

**Required configuration:** This operation can only be called if the [configuration](#tag/configuration/operation/configuration.get) for  `dailyLeaderboards.enabled` is `true`.

**CRITICAL Parameter Rules:**
- mode2 MUST be a NUMBER, not a string like "zen" or "custom"
- When mode is "time", mode2 must be one of these integers: 15, 30, 60, or 120 (representing seconds)
- When mode is "words", mode2 must be one of these integers: 10, 25, 50, or 100 (representing word count)
- Default values: page=0, pageSize=50, daysBefore=1
- DO NOT include optional parameters unless explicitly requested by the user
